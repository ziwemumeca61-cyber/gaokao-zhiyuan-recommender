怎么打开程序（Windows）
========================================

【最省事：双击启动】
1. 打开本项目文件夹（GitHub Desktop 里点 Repository → Show in Explorer）。
2. 进入 packaging 文件夹。
3. 双击 “启动应用.bat”。
4. 弹出黑色窗口，等 5~10 秒，浏览器会自动打开 http://localhost:8501 —— 这就是程序界面。

⚠️ 使用期间不要关那个黑色窗口（关了程序就停了）。用完直接关窗口即可。


【如果双击没反应 / 报错：装一次环境，以后就能一直双击】
说明：这一步是因为电脑还没装运行环境，只需装一次。

1. 安装 Python（如果没装过）
   - 到 https://www.python.org 下载并安装。
   - 安装时务必勾选 “Add Python to PATH”。

2. 安装本程序依赖（只需一次，约几分钟）
   - 打开本项目文件夹，在地址栏输入 cmd 回车，弹出黑色命令窗口。
   - 粘贴下面这行，回车：
       pip install -e ".[dev]"
   - 看到安装完成、没有红色报错即可。

3. 再次双击 packaging\启动应用.bat 即可打开。
   或在上面的命令窗口里输入下面这行也能启动：
       streamlit run app.py


【每次拉了新代码后】
- 一般直接双击 packaging\启动应用.bat 就行。
- 如果启动时提示缺少某个库（红字 ModuleNotFoundError），
  再到项目文件夹打开 cmd，跑一次：
      pip install -e ".[dev]"
  然后重新启动即可。


【常见问题】
- 浏览器没自动打开：手动打开浏览器，地址栏输入 http://localhost:8501
- 端口被占用 / 打不开：关掉旧的黑色窗口，再双击启动一次。
- 想用 AI 助手：需要在程序内的“⚙️ 数据源 / 设置”页填入 DeepSeek 密钥，
  不填也能正常用其它功能（会自动优雅降级，不报错）。

如遇其它问题，把黑色窗口里的报错文字截图发我即可。
