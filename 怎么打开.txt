怎么打开程序（Windows）
========================================

【最省事：双击启动】
1. 打开本程序文件夹。
2. 双击 “启动应用.bat”。
3. 弹出黑色窗口，等 10~30 秒（首次较慢），浏览器会自动打开
   http://127.0.0.1:8501 —— 这就是程序界面。

⚠️ 使用期间不要关那个黑色窗口（关了程序就停了）。用完直接关窗口即可。


【换台电脑第一次用，多半要装一次"运行库"】
说明：pandas 等组件依赖微软的 Visual C++ 运行库。很多电脑没装，
表现为双击后黑窗口一闪就退、或提示“启动失败”。只需装一次：

1. 下载并安装微软运行库（约 1 分钟）：
   https://aka.ms/vs/17/release/vc_redist.x64.exe
2. 装完再双击 “启动应用.bat” 即可。

（新版启动脚本会自动检测：若就是缺运行库，会直接弹出上面的下载网址，
  并把详细错误写入同目录的 “启动日志.txt”，可截图发技术支持。）


【浏览器打不开 / 显示“内网地址无法访问”】
- 优先用 Edge 或 Chrome 打开： http://127.0.0.1:8501
- 360 等浏览器有时会把本地地址当“内网”拦截：
  请用 127.0.0.1 这个地址（不要用 localhost），或换 Edge/Chrome。
- 浏览器没自动弹出：手动打开浏览器，地址栏输入 http://127.0.0.1:8501


【绿色便携版 vs 源码版】
- 便携版（推荐发给别人）：文件夹里带有 runtime 子文件夹，
  对方解压后直接双击 “启动应用.bat”，无需安装 Python。
- 源码版（自己开发用）：没有 runtime 文件夹，需要先装 Python：
  1) 到 https://www.python.org 下载安装，安装时勾选 “Add Python to PATH”。
  2) 在项目文件夹打开 cmd，运行： pip install -e .
  3) 之后双击 “启动应用.bat”，或在 cmd 里运行： streamlit run app.py


【常见问题】
- 端口被占用 / 打不开：关掉旧的黑色窗口，再双击启动一次。
- 想用 AI 助手：在程序内 “⚙️ 数据源 / 设置” 页填入 DeepSeek 密钥；
  不填也能正常用其它功能（自动降级、不报错）。
- 导出 Word/PDF：便携版已内置，直接可导；若个别环境缺库，
  会自动降级为 .rtf（Word 可开）/ 可打印网页，仍能导出。

如遇其它问题，把黑色窗口内容、或同目录的 “启动日志.txt” 截图发我即可。
