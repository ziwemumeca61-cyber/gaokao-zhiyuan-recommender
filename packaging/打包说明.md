# 打包成「开箱即用」绿色版（卖给客户用）

目标：客户拿到一个 zip，解压 → 双击 **「启动应用.bat」** → 浏览器自动打开，
**无需安装 Python、无需联网、无需命令行**。

---

## 一、怎么生成绿色版（你做一次）

需要一台**联网的 Windows 电脑**（Win10/11）：

1. 把本项目下载到这台 Windows（GitHub → Code → Download ZIP，解压）。
2. 进入 `packaging\` 文件夹，**双击 `build_portable.bat`**。
3. 脚本会自动：下载内嵌版 Python → 装好所有依赖 → 拷贝程序与数据。
4. 完成后，产物在 `dist\高考志愿推荐系统\`。
5. 把 `dist\高考志愿推荐系统\` 整个文件夹**压成 zip**，就是给客户的成品。

> 首次构建需联网下载 Python 和依赖（约几百 MB），生成的绿色版本身较大（约 400~700MB），
> 因为内含完整 Python 运行环境——这是"免安装"的代价，属正常。

## 二、客户怎么用（开箱即用）

1. 解压你给的 zip。
2. 双击里面的 **「启动应用.bat」**。
3. 等几秒，浏览器自动打开 `http://localhost:8501`，开始使用。
4. 用完关掉黑色窗口即可。**全程离线、不用装任何东西。**

## 三、做成单文件安装包 setup.exe（更专业，模板已备好）

想要客户「一路下一步」安装并自动生成桌面快捷方式：
1. 先按上面生成 `dist\高考志愿推荐系统\`；
2. 安装 **Inno Setup**（免费，https://jrsoftware.org/isdl.php）；
3. 用 Inno Setup 打开本目录的 **`installer.iss`**，点 **Build**；
4. 生成的 `高考志愿推荐系统_setup.exe` 在 `packaging\Output\`，交付客户即可。
   客户安装后双击桌面图标即启动。

> 可选：在绿色版目录放一个 `app.ico` 图标，快捷方式会用它（脚本已引用）。

## 四、贴牌（改成你/客户的品牌）

改根目录 **`branding.json`** 即可，无需改代码：
```json
{
  "app_title": "XX志愿通",
  "app_icon": "🎓",
  "subtitle": "你的slogan",
  "org_name": "XX教育",
  "disclaimer": "结果仅供参考……"
}
```
首页标题、副标题、机构署名、免责声明都会随之改变。打绿色版时该文件会一起带上。

## 四、AI 助手（可选功能）

DeepSeek AI 答疑需要密钥。两种处理：
- 不需要该功能：忽略即可，其它功能照常（会友好提示未配置）。
- 需要：在绿色版目录新建 `.streamlit\secrets.toml`，写入
  `DEEPSEEK_API_KEY = "你的密钥"`。**注意：密钥会随包分发，按你的商业策略决定是否内置。**

## 五、商业销售提醒（重要）

- **代码**：本项目代码可按其许可证商用。
- **录取/一分一段数据**：`data/` 内含你购买及整理的数据。对外销售前，请确认你拥有
  **该数据的商业再分发权利**（向数据供应商确认授权范围），避免版权风险。
- **结果免责**：界面已标注"仅供参考"。商用时建议保留并强化免责声明，
  明确最终以各省考试院与院校官方招生章程为准。

## 六、更适合长期售卖的方式：网站版（SaaS）

绿色版适合"一锤子"离线交付。若想持续运营、按账号/订阅收费、统一更新数据：
- 把应用部署到**你自己的服务器**（如阿里云/腾讯云轻量服务器），绑定域名；
- 客户用浏览器访问网址即可，**数据和更新都在你手里**，便于收费与维护；
- 也避免了 GitHub/Streamlit 海外平台在国内的网络不稳定问题。
