# gaokao-zhiyuan-recommender

基于多维数据分析与推荐算法的高考志愿填报辅助系统，支持分数线预测与个性化志愿推荐。

一个**有趣又靠谱**的高考志愿填报助手（Streamlit 应用）：用位次法 + 录取概率模型给出
冲稳保志愿表，用霍兰德兴趣测评帮你找到适合的方向，用专业百科把每个专业讲明白，
并由 DeepSeek 提供 AI 答疑。

## ✨ 功能

- 🎯 **冲稳保志愿推荐**：位次法分档 + 录取概率预测 + 兴趣匹配 + 多维综合评分
- 🧭 **兴趣测评**：霍兰德 RIASEC 测评 → 兴趣画像与适合的专业方向
- 📚 **专业百科**：每个专业讲清楚——学什么课、将来干什么、适合什么样的人
- 🔥 **热门专业推荐**：综合热度榜 + “分数够得着的热门专业”个性化版
- 🃏 **卡片选校**：像刷卡片一样挑学校，喜欢的进心愿单
- 📊 **数据大屏**：冲稳保结构、录取概率分布、兴趣雷达、位次走势
- 🤖 **AI 助手**：由 DeepSeek 提供，答疑专业与志愿
- 📄 **一键导出**：把推荐理由整理成 Markdown / Word 文档

## 🚀 快速开始

```bash
pip install -e ".[dev]"            # 安装
python data/generate_mock_data.py  # 生成模拟数据（应用也会自动生成）
streamlit run app.py               # 启动
```

可选：配置 AI 助手——复制 `.streamlit/secrets.toml.example` 为
`.streamlit/secrets.toml` 并填入 `DEEPSEEK_API_KEY`（不配置则 AI 助手降级，其余功能照常）。

## 🧪 测试

```bash
pytest -q
```
